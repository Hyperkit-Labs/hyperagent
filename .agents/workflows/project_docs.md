# Project Documentation Workflow

This workflow simply points you to the `docs/` folder where all markdown
documentation lives.

- Open `docs/README.md` for a quick start.
- Browse the individual files (`MVP‑Sprint‑Board.md`, `9‑Execution‑Strategy.md`, …) for details.

You can also render them with any markdown viewer or the built‑in
`cat` command:

\`\`\`bash
cat docs/*.md
\`\`\`
