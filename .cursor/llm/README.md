# LLM Context Documentation

This directory contains structured, AI-friendly documentation files (llm.txt format) for key technologies and libraries used in the HyperAgent platform. These files help AI agents understand, summarize, and cite project dependencies more efficiently.

## Format

All files follow the [llms.txt](https://llmstxt.org/) specification:
- H1 header with project name
- Blockquote with brief description
- H2 sections organizing content
- Markdown links to documentation
- Optional descriptions for context

## Usage

These files are automatically available to AI agents working on HyperAgent. They provide:

- Quick reference to project documentation
- Links to official documentation sites
- API references and guides
- Code examples and templates

## Adding New Documentation

To add documentation for a new project:

1. Create a new file: `docs/[project-name]-llm.txt`
2. Follow the llms.txt format:
   ```markdown
   # Project Name
   
   > Brief description of the project
   
   ## Documentation
   - [Link Title](URL): Description
   
   ## API Reference
   - [API Link](URL): Description
   ```

3. Update this README.md with the new entry
