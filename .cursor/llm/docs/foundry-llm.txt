# Foundry

> Foundry is a blazing fast, portable and modular toolkit for Ethereum application development written in Rust. Foundry consists of Forge (testing framework), Cast (Swiss army knife for interacting with EVM smart contracts), Anvil (local Ethereum node), and Chisel (fast, utilitarian, and verbose Solidity REPL).
