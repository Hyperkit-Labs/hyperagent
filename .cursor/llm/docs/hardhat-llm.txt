# Hardhat

> Hardhat is a development environment for Ethereum software. It consists of different components for editing, compiling, debugging and deploying smart contracts and dApps, all of which work together to create a complete development environment.

## Documentation

- [Getting Started](https://hardhat.org/docs): Introduction to Hardhat
- [Hardhat Network](https://hardhat.org/hardhat-network/docs): Local Ethereum network
- [Compile](https://hardhat.org/hardhat-runner/docs/guides/compile-contracts): Compiling smart contracts
- [Deploy](https://hardhat.org/hardhat-runner/docs/guides/deploying): Deploying contracts
- [Testing](https://hardhat.org/hardhat-runner/docs/guides/test-contracts): Testing smart contracts
- [Plugins](https://hardhat.org/hardhat-runner/docs/guides/plugins): Using and creating plugins
