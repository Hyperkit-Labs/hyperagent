# Supabase

> Supabase is an open source Firebase alternative. It provides a PostgreSQL database, authentication, real-time subscriptions, storage, and auto-generated APIs. Built for developers who want to build fast and ship products.

## Documentation

- [Getting Started](https://supabase.com/docs): Introduction to Supabase
- [Database](https://supabase.com/docs/guides/database): PostgreSQL database guide
- [Authentication](https://supabase.com/docs/guides/auth): Authentication and user management
- [Storage](https://supabase.com/docs/guides/storage): File storage
- [Realtime](https://supabase.com/docs/guides/realtime): Real-time subscriptions
- [Edge Functions](https://supabase.com/docs/guides/functions): Serverless functions
