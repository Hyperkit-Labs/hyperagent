# Smart Contracts

This directory contains smart contract implementations and templates.

## Structure

- **evm/**: Hardhat/Foundry projects for EVM chains
  - `contracts/`: Solidity contract source files
  - `test/`: Test files
  - `foundry.toml`: Foundry configuration
- **templates/**: Reusable contract blueprints
  - `erc20/`: ERC20 token templates
  - `erc721/`: ERC721 NFT templates
  - `staking/`: Staking contract templates

## Development

Use Foundry for EVM contracts. See individual project READMEs for setup instructions.
