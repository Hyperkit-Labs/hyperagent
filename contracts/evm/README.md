## SKALE AccountFactory

Thirdweb's shared AccountFactory is not deployed on SKALE Base Sepolia or SKALE Base Mainnet. Deploy your own thirdweb-compatible AccountFactory and configure Studio via env vars. See `docs/skale-account-factory.md`.
