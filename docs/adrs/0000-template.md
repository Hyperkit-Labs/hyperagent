# ADR NNNN: Short title in imperative mood

- **Status:** Proposed | Accepted | Superseded | Deprecated
- **Date:** YYYY-MM-DD
- **Supersedes:** (optional) ADR link or "none"
- **Superseded by:** (optional) ADR link or "none"

## Context

What is the issue or opportunity? What constraints and forces apply (product, security, ops, team)?

## Decision

What did we decide? State it in one short paragraph, then elaborate.

## Consequences

### Positive

- Bullet list of benefits.

### Negative / trade-offs

- Bullet list of costs, risks, or things we accept.

### Follow-up

- Migration steps, monitoring, or docs to update.

## References

- Links to issues, PRs, external specs, or related ADRs.
