External Secrets Operator (optional)
====================================
1. Install ESO in the cluster (https://external-secrets.io).
2. Create ClusterSecretStore pointing at AWS Secrets Manager or GCP Secret Manager.
3. Add ExternalSecret resources that sync remote keys into Kubernetes Secrets
   named orchestrator-secrets and api-gateway-secrets.
4. Use the same key names as infra/deploy/required-env.yaml (production profile).

Do not commit real remote ARNs or secret IDs.
