# Packages

This directory contains shared packages used across the monorepo.

## Structure

- **sdk-ts/**: TypeScript SDK (x402, API client)
- **ui/**: Shared React/Tailwind UI components
- **core-lib/**: Shared domain logic and types
- **cli/**: CLI tool for HyperAgent

## Usage

Packages are published and consumed internally within the monorepo using workspace protocols.
