# Agents

This directory contains agent implementations for different tasks.

## Structure

- **codegen/**: LLM pipelines for code generation
- **audit/**: Security analysis orchestration
- **deploy/**: Multi-chain deployment logic
- **spec/**: Specification refinement agent
- **monitor/**: Monitoring agent

## Agent Patterns

Each agent follows consistent patterns for:
- Input/output handling
- Error management
- Logging and observability
- State management
